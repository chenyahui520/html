## 分类

* 基本（值）数据类型

  * String
  * Number
  * boolean
  * undefined
  * null

* 引用\(对象\)数据类型

  * object:任意对象
  * Function：一种特别的对象
  * Array： 一种特别的对象（数值下标，内部数据是有序的），**数组也是对象**

## 判断数据的类型

* typeof    可以判断：undefined/数值/字符串 / 布尔值/function
  * 不能判断null和undefined  ，Object与arry
* instanceof 判断对象的具体类型
* ===   可以判断undefined null
